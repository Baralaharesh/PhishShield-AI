# PhishShield-AI
_"AI-powered phishing link analyzer and cyber security threat detector"_
**What is this projectShield 🛡️

**PhishShield** Al is a smart tool I built to protect people from online scams. You know how we often get suspicious links on WhatsApp or SMS? This app checks those links before you click them.

How does it work?

1. The Brain: It uses Google's Gemini Al to "think" like a security expert and find hidden risks in a URL.

2. The Detective: It automatically finds where the website is hosted (IP address, Country, and Company) to see if it's a fake server.
