# 🛡️ PhishShield AI: Phishing Link Analyzer

PhishShield AI అనేది ఒక అడ్వాన్స్‌డ్ సైబర్ సెక్యూరిటీ టూల్. ఇది అనుమానాస్పద లింక్‌లను (URLs) విశ్లేషించి, అవి సురక్షితమేనా కాదా అని Google Gemini AI సహాయంతో గుర్తిస్తుంది.

## 🚀 Features
* **AI-Powered Analysis:** Gemini 1.5 Flash మోడల్ ద్వారా లింక్ విశ్లేషణ.
* **Server Intelligence:** వెబ్‌సైట్ యొక్క IP Address, Country మరియు ISP వివరాలను చూపిస్తుంది.
* **Telugu Reports:** సామాన్య ప్రజలకు అర్థమయ్యేలా భద్రతా నివేదికను తెలుగులో ఇస్తుంది.
* **User-Friendly UI:** Streamlit ద్వారా తయారు చేయబడిన అందమైన ఇంటర్‌ఫేస్.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **AI Engine:** Google Gemini AI API
* **Libraries:** `google-generativeai`, `requests`, `socket`

## ⚙️ How to Use
1. ఈ యాప్ లింక్‌ని ఓపెన్ చేయండి.
2. మీ **Gemini API Key**ని సైడ్‌బార్‌లో ఎంటర్ చేయండి.
3. మీరు తనిఖీ చేయాలనుకుంటున్న URLని పేస్ట్ చేయండి.
4. **Analyze** బటన్ నొక్కి సెక్యూరిటీ రిపోర్ట్ పొందండి.
